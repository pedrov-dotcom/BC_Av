# SchoolKing Protocol

## Objetivo

MVP Web3 descentralizado desenvolvido na Ethereum Sepolia.

O protocolo integra:
- Token ERC20
- NFT ERC721
- Contrato de Staking
- DAO simplificada
- Oracle Chainlink

---

## Tecnologias Utilizadas

- Solidity ^0.8.x
- OpenZeppelin
- Remix IDE
- MetaMask
- Chainlink
- Sepolia Testnet

---

## Funcionalidades

### ERC20
Token utilitário do protocolo.

### NFT
Mint de NFTs representando conquistas.

### Staking
Stake e retirada de tokens.

### DAO
Sistema simplificado de votação.

### Oracle
Consulta de preço ETH/USD utilizando Chainlink.
